AutoMate Sale Order Line

Video Tutorial

AutoMate Sale order line module allows you to create sale order line by selecting product in sale order.

  • Creating new Sale Order Line in By selecting Product and Quantity.
  • Changing Quantity of selected product and it will reflect in sale order line.
  • Click on Create Invoice button.
  • By clicking the button it will create invoice.
  • Created invoice with Length and Total Length in invoice lines.
  • Added Length and Total Length field in Sale Order Report.
