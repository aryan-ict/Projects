Referral Program

Video Tutorial

hr_referral_application module allows you to create records in draft, approved and cancel state. You will also create record in the backend from hr_referral_application web form.

  • Can create record in the form view in any of the stage.
  • Approve Button which is on click changes the current state to approved state.
  • Create Application Button is only visible in approved state.
  • By submitting hr_referral_application web form, records will create in the module.

Prerequisites before Installation

Certain Prerequisites are required before the installation.

  • Install Employees module from Apps.
  • Install Recruitment module from Apps.
